## SMS CONFIRMATION BEHAVIOR

**What to expect:**
- Immediately after booking, you receive a confirmation text
- Inmediatamente después de reservar, recibes un texto de confirmación

- Text includes: date, time, barber name, service, shop address
- El texto incluye: fecha, hora, nombre del barbero, servicio, dirección de la tienda

- 24 hours before appointment, you receive a reminder text
- 24 horas antes de la cita, recibes un texto de recordatorio

- Text may include a link to cancel or reschedule
- El texto puede incluir un enlace para cancelar o reprogramar

**If you don't receive text:**
- Check your phone number was entered correctly
- Verifica que tu número de teléfono fue ingresado correctamente

- Check your phone can receive SMS messages
- Verifica que tu teléfono puede recibir mensajes SMS

- Check you didn't block the shop's number

# QUICK REFERENCE

## OWNER QUICK ACTIONS

**Create appointment:** New Appointment → Select client → Select barber → Select service → Pick date/time → Save
**Crear cita:** Nueva Cita → Seleccionar cliente → Seleccionar barbero → Seleccionar servicio → Elegir fecha/hora → Guardar

**Record payment:** Find appointment → Record Payment → Enter amount + tip → Select method → Save
**Registrar pago:** Encuentra cita → Registrar Pago → Ingresar monto + propina → Seleccionar método → Guardar

**Add barber:** Barbers → New Barber → Enter info → Set commission → Save
**Agregar barbero:** Barberos → Nuevo Barbero → Ingresar info → Establecer comisión → Guardar

**View reports:** Reports → Select date range → View data → Export if needed
**Ver reportes:** Reportes → Seleccionar rango de fechas → Ver datos → Exportar si es necesario

---

## BARBER QUICK ACTIONS

**Clock in:** Click "Clock In" button
**Registrar entrada:** Haz clic en botón "Registrar Entrada"

**View schedule:** Click "Today" or "My Schedule"
**Ver horario:** Haz clic en "Hoy" o "Mi Horario"

**Mark complete:** Find appointment → Mark Complete
**Marcar completa:** Encuentra cita → Marcar Completa

**Check earnings:** My Earnings → View pending commission
**Verificar ganancias:** Mis Ganancias → Ver comisión pendiente

**Clock out:** Click "Clock Out" button
**Registrar salida:** Haz clic en botón "Registrar Salida"

---

## CLIENT QUICK ACTIONS

**Book online:** Website → Book Now → Pick barber + service + time → Enter info → Confirm
**Reservar en línea:** Sitio web → Reservar Ahora → Elegir barbero + servicio + hora → Ingresar info → Confirmar

**Cancel:** Call shop → Give name + time → Request cancellation
**Cancelar:** Llamar tienda → Dar nombre + hora → Solicitar cancelación

**Reschedule:** Call shop → Give name + time → Request new time
**Reprogramar:** Llamar tienda → Dar nombre + hora → Solicitar nueva hora
